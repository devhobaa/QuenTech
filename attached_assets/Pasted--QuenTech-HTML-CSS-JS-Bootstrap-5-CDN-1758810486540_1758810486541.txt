ابنِ لي موقع احترافي كامل لشركة برمجيات باسم QuenTech باستخدام HTML + CSS + JS + Bootstrap 5 عبر CDN، مع دعم لغتين (العربية والإنجليزية) وإمكانية التبديل بينهما. اجعل التصميم عصري وكرييتف، واستخدم FontAwesome للأيقونات، وأضف animations عند السكول باستخدام مكتبة AOS أو أي مكتبة JS خفيفة.

الصفحات والمميزات:
1. الصفحة الرئيسية (Home Page):

Hero Section: صورة أو سلايدر جذاب مع شعار الشركة + زر اطلب خدمتك الآن.

تعريف قصير بالشركة ورسالتها.

أقسام الخدمات: (بناء مواقع – متاجر إلكترونية – منصات تعليمية – برامج سطح مكتب – تطبيقات موبايل – حلول خاصة) مع أيقونات FontAwesome.

لماذا نحن (Why Us): مع أيقونات توضيحية.

آراء العملاء (Testimonials).

تواصل معنا (Contact Form)، يتم إرسال البيانات مباشرة إلى Supabase.

2. صفحة الخدمات (Services Page):

تفاصيل كل خدمة مع صور/أيقونات واضحة.

انميشن عند السكول لكل عنصر خدمة.

3. صفحة الطلب (Order Page):

نموذج إدخال بيانات العميل: الاسم، الإيميل، رقم الهاتف، نوع الخدمة، تفاصيل إضافية.

رفع ملفات اختيارية.

زر إرسال يحفظ البيانات مباشرة في Supabase باستخدام Supabase JS SDK.

إضافة رسالة نجاح/فشل بعد إرسال النموذج.

4. صفحة من نحن (About Us Page):

نبذة عن الفريق، الرؤية، القيم.

صور الفريق مع أيقونات FontAwesome للتواصل.

5. Footer:

روابط سريعة، حقوق الملكية، أيقونات السوشيال ميديا.

التصميم والستايل:

Bootstrap 5 (CDN) مع تخصيص CSS لجعل التصميم عصري وكرييتف.

ألوان حديثة وإبداعية (تدرجات أزرق وبنفسجي أو ألوان مميزة).

FontAwesome للأيقونات.

Animations عند السكول: استخدم مكتبة AOS (Animate On Scroll) أو مكتبة JS خفيفة.

أزرار وكروت أنيقة مع تأثيرات hover.

الموقع متجاوب بالكامل (Responsive).

المتطلبات التقنية والمميزات الإضافية:

الموقع HTML + CSS + JS فقط مع Bootstrap CDN، لا حاجة لـ React أو Node.js.

i18n للغات باستخدام JS بسيط (تبديل النصوص بين العربية والإنجليزية).

Supabase Integration:

جميع النماذج (Contact Form وOrder Page) ترسل البيانات مباشرة إلى Supabase.

استخدام Supabase JS SDK للاتصال بقاعدة البيانات.

تضمين استدعاءات API لإنشاء / قراءة الطلبات.

إضافة رسائل تأكيدية للمستخدم عند نجاح أو فشل الإرسال.

التركيز على تجربة المستخدم (UX) وسهولة الاستخدام.

كل عنصر يجب أن يكون في component منفصل قدر الإمكان (HTML جزئي أو Sections).

اجعل الكود مرتب وواضح مع التعليقات لتسهيل التعديل لاحقًا.

لو تحب، أقدر أصنع لك نسخة جاهزة كاملة تحتوي على:

HTML لكل الصفحات (Home – Services – Order – About)

CSS مخصص + Bootstrap

JS لتبديل اللغات + Animations عند السكول + اتصال بـ Supabase

FontAwesome للأيقونات

Footer متكامل

وهتكون جاهزة للتشغيل مباشرة على المتصفح مع تخزين البيانات في Supabase.
